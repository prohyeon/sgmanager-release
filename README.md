# SGManager Releases

This repository is the public distribution channel for SGManager.

The source code is maintained in a private development repository. This public
repository only contains release notes and downloadable release assets.

## Download

Use the latest GitHub Release to download `SGManager.exe`.

## Update Channel

SGManager can check this repository's latest release endpoint:

```text
https://api.github.com/repos/prohyeon/sgmanager-release/releases/latest
```

Release assets are attached to each versioned GitHub Release.
